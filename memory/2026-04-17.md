## Architectural Decisions

### 1. **Raul API Integration Pattern**
- Created separate `raul_integrator.py` module to prevent tight coupling
- Fetches BYND context from Raul's `/api/tickers?window=full` endpoint
- Merges institutional data (L/S ratio, bias, outfit, key SMAs, active signals) into monitor output
- Called from monitor.py before data.json write
- **Pattern Value:** Modular integration; Raul API failures don't crash monitor

### 2. **Exit Signal Logic (Multi-Layer)**
- Combine monitor squeeze state + institutional bias for actionable signals
- Example: FADING (no vol expansion) + BEARISH bias = strong institutional exit signal
- Dashboard red alert box highlights when both conditions met
- **Pattern Value:** Removes ambiguity; single-layer state not enough for real exit confidence

### 3. **Data Structure: Nested Institutional Context**
- data.json now includes nested `raul_context` object:
  ```json
  {
    "raul_context": {
      "rank": 1,
      "total_weight": 33637.9,
      "ls_ratio": 1.29,
      "institutional_bias": "Neutral",
      "dominant_outfit": "REPEATER_9",
      "key_smas": ["MA54", "MA250", "MA200", "MA45", "MA246"]
    },
    "watch_for": {
      "outfit_change": "REPEATER_9 → new outfit = position shift",
      "bias_flip": "NEUTRAL → BEARISH = exit signal strengthens",
      "ls_decline": "< 1.2x = conviction loss, watch support breaks"
    }
  }
  ```
- Dashboard renders institutional panel + alerts dynamically
- **Pattern Value:** Single JSON source-of-truth; no API call duplication

## Bug Fixes

### Price Formatting for Penny Stocks
**Problem:** BYND $0.82 was displaying as $0.8200 (wrong decimal precision)
**Solution:** Smart decimal logic based on price tier:
- Subpenny (<$0.01): 4 decimals
- Penny ($0.01–$1): 2 decimals → $0.82
- Normal ($1–$100): 2 decimals → $15.45
- Large (>$100): 0 decimals → $245

**Impact:** Dashboard now readable for low-price securities; critical for BYND tracking

## Key Lessons

1. **Separate integrator modules prevent coupling:** Raul API integration isolated from core monitor logic; can upgrade/debug independently

2. **Multi-layer exit signals reduce false positives:** FADING alone = might be consolidation; FADING + BEARISH bias = institutional patience exhausted, real exit signal

3. **Nested data structures > flat JSON:** Institutional context grouped together; easier to extend without breaking dashboard rendering

4. **Price formatting details matter:** Visual credibility depends on correct decimal places; looks sloppy at $0.8200 when ticker trades $0.82
