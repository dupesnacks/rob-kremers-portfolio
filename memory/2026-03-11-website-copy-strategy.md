# Website Copy Strategic Decision - March 11, 2026

## MAJOR DECISION: Emotional-First Website Redesign for sensorygalaxy.com

### Context
Rob asked me to review the landing page copy and create a proposal for reframing it to be more emotionally resonant (like the onboarding redesign) to maximize trial signup conversion.

### Key Insight: Current Site is Built for Believers
The current website assumes people already know they want the app. It explains features. But it doesn't convince people they NEED it.

### The Strategic Shift

**FROM:** Feature-first ("Turn picky eating into an adventure")  
**TO:** Emotion-first ("You've tried everything. They still eat 5 foods.")

### Why This Works (Based on Arise Case Study)
Arise: 13% → 23% conversion by designing for emotional recognition, not features.  
Apply same principle to website: parent sees themselves → pain validated → solution offered → low-risk trial

### Proposed 8-Section Structure

1. **Hero** - Recognition hook (pain point upfront)
2. **Testimonials** - NEW section for social proof
3. **How It Works** - Explain 8 planets + SOS method
4. **Explorers** - Position as mechanism for real change
5. **Features** - Reframe as 3-part flow (Understand → Suggest → Celebrate)
6. **Why Parents Love** - Benefits not features
7. **Pricing** - Risk-free, reassurance-focused
8. **Final CTA** - Big button before footer

### Key Changes

**Headlines:**  
- "Turn picky eating into an adventure" → "You've tried everything. They still eat 5 foods."
- "Meet your explorers" → "Make it fun: Your child's explorer"
- "Smart analytics" → "Understand the pattern"

**Tone:**  
- From: Corporate, clinical, feature-focused
- To: Warm, honest, parent-first, hopeful

**Content Additions:**
- 3 parent testimonials (names, locations, real quotes)
- Stat: "1 in 3 kids" (normalize problem)
- Reassurance copy: "No credit card," "Cancel anytime," "See if it works for YOUR family"
- Expectation-setting: "Most parents see progress by week 3"

**Imagery Needs:**
- NEW Hero image: Parent + kid, warm lighting, hopeful (replace Nova explorer headshot)
- 3 testimonial photos (real people or stock)
- Keep: All planets, explorers, color palette

### Predicted Impact
- Current conversion: ~2-3% (hero → trial signup)
- Predicted with changes: 5-8% (3x improvement)

### Rollout Process
1. Rob reviews proposal
2. Rob marks up: love/hate/adjust/add
3. Rob tells me: testimonials (real or stock)?
4. Rob sources imagery when ready
5. I integrate copy + imagery + deploy
