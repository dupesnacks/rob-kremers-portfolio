**CEA Fix:** Rayleigh phase coherence
- Circular statistics: measures phase alignment of 3 cycles

**Use Case:** LPI (3-5 day surge prediction)

**Lesson:** Higher-order derivatives capture momentum changes

## 💡 Technical Insights Discovered

### 1. **Why Astro's 232% CAGR Works**
- Not ML magic or hidden indicators
- Pure **deterministic cycle analysis** + **self-fulfilling prophecy**
- If enough traders follow celestial rhythms, they become real
- Edge lasts as long as information asymmetry (few know the secret)

### 2. **Why Fake Backtests Mislead**
- Circular validation: testing signal against same data that generates signal
- Real markets have *friction* (slippage, fees, liquidity) that backtests skip
- Proving > hodling in good environments ≠ proving true alpha

### 3. **Why Adaptive Thresholds Matter**
- Volatility is **regime indicator**
- Low vol = clear trend = tight rules work fine
- High vol = noise = loose rules prevent whipsaws
- Fixed rules = brittle

### 4. **Why Phase Coherence is Better Than Indicators**
- Indicators lag (by definition, they react to price)
- Phase coherence predicts (by definition, it projects)
- Circular statistics captures cyclical behavior better than linear methods

### 5. **Why Regime Detection is Critical**
- Different rules work in different markets
- Bull trending ≠ bear trending ≠ chop
- REGIME_CHANGE is danger zone (reduce confidence)
- Same signal means different things in different regimes
