- No export option in UI
- Solution: Browser DevTools Console JavaScript extraction

### Extraction Plan
**Best method: DevTools Console**
1. User logs into service.tesla.com
2. Navigate to correction codes page
3. Open DevTools (Cmd+Option+I)
4. Run: `copy(JSON.stringify(Array.from(document.querySelectorAll('table tr')).map(row => ({code: row.cells[0]?.textContent, description: row.cells[1]?.textContent})), null, 2))` (or variant based on actual HTML structure)
5. Paste extracted JSON to me
6. I parse into Excel + RAG chunks

## New Pattern: Authenticated Web Scraping
When facing blocked public APIs:
1. Try public endpoints first (fail gracefully)
2. If user has auth access, use browser DevTools extraction
3. JavaScript in Console can extract table data in seconds
4. Better than Selenium/Playwright for one-off extractions
5. Works for any HTML table structure
