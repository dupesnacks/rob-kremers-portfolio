# Smart Home Dashboard

Custom unified control dashboard for Myron's smart home ecosystem.

## Stack

- **Backend:** Node.js/Express
- **Frontend:** React + Vite
- **Deployment:** Vercel
- **APIs:** Alexa Smart Home, Wyze, Google Nest, Roborock

## Devices Integrated

- Nest Thermostat & Lock
- Roborock Vacuum
- Wyze Cameras, Vacuum, Scale
- Alexa-connected lights & power strips
- Custom automations & Siri Shortcuts

## Project Structure

smart-home-dashboard/
