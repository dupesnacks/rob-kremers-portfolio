# Sensory Explorer Badge Design Spec

**Badge System Overview:**
- 15-20 unique badges tracking food exploration journey
- Tracked in app (not "worn" on character)
- Some labeled as "RARE" for special challenges
- Kid-friendly names related to sensory food journey
- Fully compatible with app brand (space/galaxy theme + sensory language)

---

## Badge Specifications

### Visual Requirements
- **Format:** PNG with transparent background
- **Size:** 256×256px (icon size for app display)
- **Style:** Cartoon illustrated, rounded edges, warm lighting
- **Brand Colors:** Cyan #00d9ff, Gold #d4af37, Purple #667eea, Pink #e91e63
- **Aesthetic:** Fun, achievement-focused, space/explorer themed

---

## Implementation Notes

### Size Standards for App Integration
- **List Display:** 128×128px (scaled down from 256×256 original)
- **Detail View:** 256×256px (full size)
- **Tooltip/Preview:** 64×64px (scaled down)

### Storage & Naming
- Store all badges at 256×256px original
- File naming: `badge-01-first-taste-explorer.png` through `badge-20-galaxy-legend.png`
- Organize in `/public/badges/` folder

### UI Placement (in Settings/Cosmetics tab)
```
Achievement Badges (X/20)
[Grid view showing unlocked badges]
[Locked badges appear with lock icon]

Filter by:
- [ ] COMMON
- [ ] UNCOMMON  
- [ ] RARE
- [ ] LEGENDARY
```

### Parent Dashboard Note
Show in parent dashboard:
- Total badges earned (X/20)
- Recent badges unlocked
- Next badges to work toward (top 3)
- Timeline of badge unlocks (when earned)
